# История изменений

## [1.0.0-etalon] — 2026-07-11

### Эталонная стабильная сборка

#### GUI
- Минималистичный светлый интерфейс WPF на русском языке
- Карточки статистики с быстрым фильтром по клику
- Панель поиска, фильтры направления и статуса
- Заголовки колонок в списке правил
- Модальное окно создания правила
- Пустые состояния: нет правил / ничего не найдено
- Кнопки действий отключаются без выбора правила
- Горячие клавиши: Ctrl+N, Ctrl+F, F5, Delete, Esc
- Двойной клик — вкл/выкл правила

#### Правила
- TCP, UDP, ICMPv4/v6, GRE, IGMP, SCTP, ESP, AH, TCP+UDP, Любой
- Фильтрация по удалённым и локальным IP

#### Исправления
- Раскладка Grid без перекрытий
- Вкладки фильтров без обрезания текста
- BrushConverter вместо Brush.Parse
